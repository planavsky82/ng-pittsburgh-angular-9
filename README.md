
## Install Demo

- `git clone https://github.com/planavsky82/ng-pittsburgh-angular-9.git` 
- `cd ng-pittsburgh-angular-9 && python -m SimpleHTTPServer 8000`
- Open in browser: [http://localhost:8000](http://localhost:8000)
